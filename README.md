# 我的第一个AI产品构想：HR简历初筛助手

## 1. 背景与问题
作为HR，我每天收到几百份简历，80%都不匹配，手动筛选耗时耗力。

## 2. 用户故事
作为一个招聘HR，我希望AI能帮我**自动阅读简历并提取关键信息**，并**判断与岗位的匹配度**，以便我能专注于前20%的候选人。

## 3. 核心功能 (MVP - 最简可行产品)
- 功能一：简历上传（支持PDF/Word）。
- 功能二：关键信息提取（姓名、电话、工作年限、技能关键词）。
- 功能三：匹配度打分（基于岗位描述JD）。
- 功能四：标注疑点（例如：存在1年以上职业空窗期）。

## 4. 数据需求
需要至少500份已标注的简历-JD配对数据来训练模型。初期可由HR手动标注。

## 5. 成功指标
- 准确率：匹配度判断准确率达到80%。
- 效率：HR筛选100份简历的时间从3小时缩短到30分钟。
