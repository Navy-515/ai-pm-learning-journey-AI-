# 🏦 机构投研助手 · AI 股票分析师

> 基于 Coze 平台 + 9 插件协同构建的 AI 股票研究智能体。输入股票代码，自动生成专业投研报告。

## 解决什么问题

一线公募基金研究员每天 80% 的时间花在数据收集、整理和初步分析上。**机构投研助手**的目标就是自动化这 80% 的重复工作，让研究员把精力集中在更深度的判断和决策上。

## 设计思路

### 角色驱动
系统设定为"持有 CFA 执照的资深股票分析师"——让 AI 的输出风格、分析深度、专业术语都符合机构投资者的预期。

### 四步工作流
1️⃣ 获取基础数据 → 2️⃣ 调用分析插件 → 3️⃣ DCF 估值决策 → 4️⃣ 输出结构化研报

### 插件协同架构
- **2 个官方插件**：股票查询(search_code_info)、新浪财经(SearchStockData)
- **7 个自建插件**：基本面分析、历史趋势、同业对比、护城河分析、财务健康诊断、DCF 估值、财报 PDF 解析

### 强制数据溯源
回答中的所有具体数据必须来自对应插件的返回结果，禁止编造。如果插件返回异常，必须明确标注。

## 使用示例

### 示例 1：查询贵州茅台

**用户输入：** `600519`

**输出结构：**
- 📊 标的概况（实时行情 + 公司信息）
- ✨ 投资亮点（护城河 AAA、ROE 34.2%、CAGR 14.1%）
- ⚠️ 风险提示（财务健康、估值偏高、同业竞争）
- 💰 估值判断（DCF 每股价值 1,682.50 元，安全边际 +19.1%）
- 🎯 综合评级：观望

### 示例 2：分析宁德时代

**用户输入：** `300750`

**执行流程：** 调用官方插件 → 基本面分析 → 同业对比 → 护城河分析 → 财务健康 → DCF 估值 → 输出完整研报
