# 🌐 个人作品集网站

> 展示所有 AI 产品项目的统一门户网站。

## 页面结构
- **首页**：两个核心项目（Coze 投研助手 + Dify 客服助手）导航卡片
- **机构投研助手**：9 插件协同架构、完整提示词、四步工作流、测试输出
- **7 个自建插件详情**：每个插件的完整功能说明、参数规格、评级体系
- **HashKey Exchange 客服**：父子分段知识库、双层检索、问答测试记录

## 技术说明
- 纯 HTML + CSS 静态页面，无框架依赖
- Google Fonts (Inter) 使用国内镜像 CDN `fonts.loli.net`
- 支持 Netlify Drop 一键部署，拖拽文件夹即可更新
